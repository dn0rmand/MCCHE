Motion Two
1) Instead of setting down screens, we will set back screens for layups.  The ball handler will yell out, "TWO" to initiate the play.

2) The post players will set screens for the wings.

3) The screens are set; only then should the wings break towards the basket.

4) The ball handler may pass to either cutter for a layup if open.  Otherwise, he can reset and run the normal motion offense.

5) Another variation would be "Motion L-word" where the left side sets a back screen and the right side sets a down screen.  "Motion R-word" has the right side set the back screen and the left side set the down screen.
